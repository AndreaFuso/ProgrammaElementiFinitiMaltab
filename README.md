# Programma Elementi Finiti Maltab
Questo programma permette di calcolare gli spostamenti nodali di uan struttura meccanica semplice 2D, soggetta a dei carichi esterni quali: carichi distribuiti (assiali,trasversali e momenti), forze di volume, forze di superficie, forze nodali e deformazioni termiche delle bielle.
